Problem Statement :

The project is designed to create a billing system for a supermarket where users can select items from a predefined list, input quantities, and get a detailed bill with prices, GST calculation, and the final amount. It aims to automate the billing process for more efficiency and accuracy.


Scope of the Project :


This project handles billing for a fixed list of supermarket items with their prices. It calculates totals, applies GST, and generates a formatted bill including customer name and date. It does not cover inventory management, user authentication, or payment processing.



Target Users :



Small supermarket owners or cashiers who want a simple billing tool.

Beginners learning Python programming by building real-world projects.

Educational purposes to understand basic input handling, calculations, and printing formatted output.


High-level Features :


Display predefined list of items with prices.

Input multiple items and quantities to add to the bill.

Calculate total price and GST (5%).

Generate and print a detailed bill with customer name and date.

Validate item availability and offer option to exit billing.
